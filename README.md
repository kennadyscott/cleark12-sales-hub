# ClearK12 Sales Hub

Sales enablement web platform for ClearK12 — brand messaging + sales playbook for **Crystal Instruction** and **LoneStar CR**.

## Quick start (in Claude Code)
1. Open this folder in Claude Code.
2. Say: **"Read CLAUDE.md and SPEC.md, then scaffold the v1 platform."**
3. Add your three logo PNGs to `assets/` (see `assets/README.md`).

## What's here
- `CLAUDE.md` — context + working rules for building in this repo.
- `SPEC.md` — full build spec: stack, routes, features, theming, definition of done.
- `content/content.json` — all sales copy (single source of truth; the UI renders this).
- `assets/` — drop logos here.

No app code yet — this is a content + spec handoff, ready to build from.
